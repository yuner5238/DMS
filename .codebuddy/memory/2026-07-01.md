# 2026-07-01

## 移动端 header-row 响应式优化（三轮）
- 根因：`@media (max-width: 1024px)` 下 `.header-row` 变为 `flex-direction: column`，而 `.search-container` 的 `flex: 1 1 180px` 在列方向下 flex-basis 作用于高度，导致搜索框至少 180px 高。

### 第一轮修复（搜索框高度 + toolbar 不换行）
- `.search-container { flex: 0 0 auto; }` — 高度由内容决定
- `.sidebar-toggle-btn { align-self: flex-start; }` — 防止 hamburger 被 stretch 到全宽
- `.toolbar { margin-left: 0; justify-content: flex-start; }` — 列布局下左对齐
- `@768px .toolbar { flex-wrap: nowrap; gap: 4px; }` — 按钮不换行
- `@768px .toolbar-btn { padding: 5px 10px; gap: 3px; font-size: var(--text-xs); }` — 收紧间距

### 第二轮优化（断点分层 + 平板横向保留）
- **列布局从 1024px 下沉到 768px**：平板(769-1024px) 保留横向 header-row，只收紧间距
- `@1024px`：content-header padding 32→20px，toolbar 按钮稍缩小(5px 12px)，gap 收至 5px
- `@768px`：列布局激活，header-row gap 12→8px，其余移动端卡片布局不变
- 桌面端(>1024px) 完全不受影响

### 第三轮优化（移动端搜索栏 + 筛选按钮）
- 在 `.search-container` 内新增 `#searchFilterBtn`（`bi-sliders` 图标），默认隐藏，仅在 `@768px` 显示
- 移动端隐藏 toolbar 中原 `#sortBtn`，由搜索栏右侧的筛选按钮统一触发排序下拉菜单
- `#sortDropdown` 从 `.toolbar` 内移到 `.header-row` 层级，JS 动态计算相对点击按钮的 `top/left`
- `#searchInput` 固定高度 36px，与筛选按钮等高，视觉统一
- 桌面端完全不受影响

### 导入弹窗按钮事件不一致修复
- **问题**：导入弹窗默认显示"批量粘贴"标签页，但"开始导入"按钮的 `onclick` 一直指向文件导入函数 `doImport()`，导致在批量粘贴 tab 中点击后提示"请选择文件"
- **根因**：`openImportModal()` 中初始 HTML 将"批量粘贴"tab 置为 active 并显示 paste panel，但没有调用 `switchImportTab('paste')` 来同步更新按钮事件绑定
- **修复**：在 `openImportModal()` 末尾添加 `switchImportTab('paste')`，确保按钮 handler 与当前激活的 tab 一致

### 三合一功能更新（所属仓库列 + 序列号悬停预览 + 标签 tooltip 裁剪修复）
- **新增"所属仓库"列**：COLUMN_DEFS 中在 name 后插入 `{ key: 'warehouse', label: '所属仓库', dataCol: 'warehouse' }`，默认可见。`renderTable()` 的 cellMap 新增 `warehouse` 单元格，无仓库时显示"未指定"。CSS 添加 `th[data-col="warehouse"] { width: 5%; }`。COLUMN_VERSION 4→5，columnOrder 初始化逻辑改为按 COLUMN_DEFS 顺序插入新 key（而非追加到末尾）
- **序列号悬停预览**：`serialNumber` 单元格添加 `title` 属性，鼠标悬停显示完整序列号
- **标签 "+n" tooltip 被 overflow:hidden 裁剪修复**：根因是 `td[data-col="tags"] { overflow: hidden }` 将 `.tag-more-tip`（position:absolute）裁剪。修复：添加 `td[data-col="tags"]:has(.tag-more:hover) { overflow: visible; }`，仅在悬停 "+n" 时允许溢出
