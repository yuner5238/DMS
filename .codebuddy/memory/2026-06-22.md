## 2026-06-23 — v4 Linear 完成功能实现
- **核心问题**：原 v4-linear/index.html 引用了 `../js/index.js`(共享 JS)，该 JS 依赖 Bootstrap Modal API 和 Bootstrap 表单类，与 v4-linear 的自定义 CSS 模态框完全不兼容。所有按钮实际上无法工作。
- **解决方案**：创建完全自包含的 `v4-linear/js/app.js`，无 Bootstrap 依赖，纯 DOM+API 操作。
- **app.js 实现功能**：
  - 仓库管理：列表/添加/编辑/删除（侧边栏 + 弹出层）
  - 设备 CRUD：添加/编辑/删除（动态表单生成）
  - 出库/入库：弹出确认表单
  - 统计卡片：点击切换全部/在库/出库/临期筛选
  - 侧边栏：仓库列表、临期提醒、标签筛选
  - 导出：Excel 下载
  - 导入：文件上传解析
  - 排序：名称/型号/仓库/入库时间/到期 多字段排序
  - 搜索：实时过滤
  - 公告：顶栏 + 弹出查看
  - Toast 通知：成功/错误/警告/信息
  - 键盘：Esc 关闭所有模态框
- **device.html 修复**：`openRichTextEditorForDevice` 不再打开旧版共享 JS 页面，改为直接聚焦备注文本框
- **CSS 补充**：添加 `.tag-item-inline`、`.image-fullscreen-overlay` 样式
- **index.html** 引用从 `../js/index.js` 切换为 `js/app.js`
